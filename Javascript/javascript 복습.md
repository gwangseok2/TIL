## [switch]

```javascript
switch문에서는 if문에서의 else와 같은 default가 맨 위에 있어도 된다.

switch (val) {
	default:
        console.log('뭐고');
        break;
	case: '1' :
        console.log('1');
        break;
    case: '2' :
    	console.log('2');
        break;
}
```
